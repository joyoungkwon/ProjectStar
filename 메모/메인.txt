#별다별 - 리뷰 모아보기

○ 페이지
1) 회원관리 - 로그인, 회원가입
2) 리뷰 게시판
3) 리뷰 상세보기
4) 리뷰 작성
5) 마이페이지
6) 문의페이지

○ 회원관리
1) 카카오 API

○ 리뷰 게시판
1) 출력 방식 - 미리보기/텍스트
2) 검색 설정 - 키워드, 위치(마커), 상호명, 유저명
3) 정렬 - 최신순, 조회순, 추첨순, 별점순
4) 신뢰도 - 작성한 유저의 신뢰온도(작성한 리뷰의 추첨/비추첨 수에 따라 점수 부여) 출력

○ 리뷰 작성
1) 내 지역만 리뷰 작성 가능
2) 정확한 리뷰 위치 설정 (마커/검색)
3) 사진/영상 업로드, 텍스트작성, 태그설정(태그 선택 or 직접 쓰기), 별점선택
4) 로그인 필수

○ 마이페이지
1) 리뷰/유저의 즐겨찾기 목록

○ 문의페이지(시간 남으면 구현)
1) FAQ
2) 1:1문의
3) 신고내역

○ 모바일(시간 남으면 구현)
1) 반응형 웹
2) 앱.
